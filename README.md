# Java Swing - Pong game

Hra vytvořená pomocí Java Swing. Hra pro dva hráče, každý hráč vidí své skóre, míček po každém odrazu zrychlí. Pokud hráč neodrazí míček, hra automaticky spustí nové kolo.
